Boot-vibe placements

Your splash stays as-is. These are the other spots the terminal-boot look could live. Tap Replay on each to see it in context.

A · Post-splash Boot GateLaunch
Right after your splash, a quick boot log types out, hits READY, then wipes to the dashboard. Splash untouched — this is the hand-off moment.
B · Empty StatesLedger/Bills/Goals
One shared helper feeds Ledger, Bills & Goals — so this single change shows up in all three when there's nothing yet.
C · Status ToastsApp-wide
Every save confirmation (add expense, save plan, etc.) slides up as a mono terminal line in accent green.
D · PIN / Lock ScreenAuth
The unlock screen becomes a terminal access prompt — dots fill as you type, then ACCESS GRANTED ✓ (or a red DENIED glitch).
E · Net Worth RevealAccounts
When the accounts overview opens, the Net Worth figure scrambles through digits and locks in (the Datastream effect).
F · Update CheckSettings/Accounts
Tapping "Check for update" prints a live boot log — checking remote, version found / up to date.
